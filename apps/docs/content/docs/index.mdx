# Media SDK

## Quick Start

<Steps>

### Install the package

```bash
npm install @jamesaphoenix/media-sdk
```

### Import and create a timeline

```typescript
import { Timeline } from '@jamesaphoenix/media-sdk';

const timeline = new Timeline()
  .addVideo('input.mp4')
  .addText('Hello, World!', {
    position: { x: 'center', y: 'center' },
    style: { fontSize: 48, color: '#ffffff' }
  })
  .setDuration(10);
```

### Generate FFmpeg command

```typescript
const command = timeline.getCommand('output.mp4');
console.log(command);
// Outputs: ffmpeg -i input.mp4 -filter_complex "..." output.mp4
```

### Execute the command

```bash
# Run the generated command
ffmpeg -i input.mp4 -filter_complex "..." output.mp4
```

</Steps>

## Core Concepts

### Timeline API

The Timeline is the core abstraction in Media SDK. It represents a video composition that can include:

- **Videos** - Base media files
- **Audio** - Background music or voiceovers  
- **Text** - Overlays with rich styling
- **Images** - Logo overlays and graphics
- **Filters** - Color grading and effects

### Functional Composition

Media SDK uses functional programming patterns:

```typescript
const result = timeline
  .addVideo('input.mp4')
  .pipe(t => t.addFilter('brightness=1.2'))
  .addText('Title', { startTime: 2 })
  .setAspectRatio('16:9');
```

### Platform Presets

Built-in optimizations for social media platforms:

```typescript
// TikTok optimized (9:16, mobile-first)
const tiktok = new Timeline()
  .setAspectRatio('9:16')
  .setResolution(1080, 1920);

// YouTube optimized (16:9, desktop-first)  
const youtube = new Timeline()
  .setAspectRatio('16:9')
  .setResolution(1920, 1080);
```
